## Getting Started

Simply Open the App or Feature you wish to learn by selecting a folder of your choosing.

1. Clone the repo

```sh
git clone https://github.com/FalseG0d/AdvancedDjango.git
```

2. Install Django Package

```sh
pip install django
```
